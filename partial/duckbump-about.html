About Buckbump

Development

Duckbump was developed using the Java programming language and the Greenfoot framework. The font for the game and website come from Joiro Hatagaya's 8-BIT font. Development of Duckbump started early September, 2014.

Future Work

While the game is in early stages of development, there is high hope for continued development and game advancement. By the end of the year, we hope to have a fully functioning android version of the game, with hopes of an ios version to follow.
